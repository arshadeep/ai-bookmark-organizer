Gemini API Configuration

Your API key is stored locally on your device and is never sent to our servers.

How to Get a Gemini API Key

  1. Go to Google AI Studio
  2. Sign in with your Google account
  3. Go to the API keys section
  4. Create a new API key
  5. Copy and paste the key here

The extension needs this API key to categorize your bookmarks using Google's Gemini AI.

About AI Bookmark Organizer

This extension automatically categorizes your bookmarks using Google Gemini AI.

When you bookmark a page, the extension will:

  • Analyze the page content
  • Suggest the best folder based on the content
  • Save the bookmark in the suggested folder or create a new one
