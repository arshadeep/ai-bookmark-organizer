How to Get a Gemini API Key

  1. Go to Google AI Studio
  2. Sign in with your Google account
  3. Navigate to the API keys section
  4. Create a new API key
  5. Copy and paste the key above

What this feature does:

About AI Bookmark Organizer

This extension automatically categorizes your bookmarks using Google Gemini AI and learns from your organization patterns.

When you bookmark a page, the extension will:
  • Analyze the page content intelligently
  • Learn from your existing bookmark organization
  • Suggest the best folder based on content and your patterns
  • Save the bookmark in the suggested folder or create a new one
